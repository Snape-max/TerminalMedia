## Installation

```bash
pip3 install dspi
```

or

```bash
git clone https://github.com/Snape-max/dspi.git
cd dspi
python install .
```

## Requirements

- Python 3.6+
- Pillow for image reading


## Usage

```bash
dspi <image_path>
```
